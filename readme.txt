== Description ==
Responsi is a lightweight Framework that sits on top of core WordPress to make every aspect of WordPress theme design completely customizable with instant live preview. Install and activate Responsi Framework and it opens the WordPress Customizer menu with its 700+ site layout and style options. Responsi Framework empowers everyone regardless of skill level or experience to create a unique hand crafted WordPress theme for their site. 

= Framework Features =
* The lightweight framework that makes every aspect of core WordPress layout and style customizable
* Responsi Customizer Menu is logically sorted into the individual elements of a WordPress theme
* 700+ Layout and style options in the WordPress Customizer menu
* Menus feature conditional logic - options that are not applicable are not shown on the menu
* Full support for widgets shift-click editing
* View all layout and style edits live in the customizer preview screen as you do them
* Cross platform code for great display in mobile, tablet, Laptop and PC
* Shema.org tags built into the framework
* No child theme required (however Free Responsi Child themes are included)
* Open Source code with Public Github repository
* Framework is endlessly extendable with features added via Free and Premium Responsi Plugins and Themes


== Installation ==

1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. Type in Responsi in the search form and press the 'Enter' key on your keyboard.
3. Click on the 'Activate' button and you will be redirected to the Responsi WordPress Customizer menu
4. If this is a new site install the Responsi Dummy Data plugin and run it
5. Go to Appearance > Responsi Plugins and install and activate the Responsi Plugins that you require - example Resonsi Widgets
6. Go to Appearance > Responsi Themes and install and activate any of the Free Responsi Child Themes
